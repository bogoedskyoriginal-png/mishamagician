# Магическое предсказание — инструкция

## Вариант 1 — PHP-хостинг (через Менеджер файлов)
1. Откройте менеджер файлов хостинга и перейдите в `public_html`.
2. Загрузите **содержимое папки `public`** из проекта (не саму папку, а файлы внутри).
3. Убедитесь, что папка `data` существует и доступна для записи.
4. Включите поддержку PHP (обычно по умолчанию).

Теперь сайт работает без Node.js.
Если хостинг не поддерживает `.htaccess`, можно открывать страницы так:
- Зритель: `https://ВАШ_ДОМЕН/index.html?slug=ВАШ_SLUG`
- Админка: `https://ВАШ_ДОМЕН/admin.html?slug=ВАШ_SLUG`
 - Главная админка: `https://ВАШ_ДОМЕН/crp.html`

## Вариант 2 — Node.js (локально или VPS)
1. Установите Node.js: https://nodejs.org/
2. Откройте терминал в папке проекта.
3. Установите зависимость:
   ```bash
   npm install express
   ```
4. Запустите сервер:
   ```bash
   node server.js
   ```

Сервер по умолчанию слушает порт `8080` (можно поменять через `PORT`).

Создаётся автоматически:
- Зритель: `https://ВАШ_ДОМЕН/default`
- Админка: `https://ВАШ_ДОМЕН/a/default`

## Как пользоваться (админка)
1. Откройте свою админ-ссылку `/a/ВАШ_SLUG`.
2. Нажмите ⚙ и задайте 5 предметов.
3. Зажмите экран на 3 секунды.
4. Тапните 1–5 раз — отправится выбор.
5. Для сброса нажмите кнопку «Сброс».

## Как открыть страницу зрителя
Зрителю отправляется ссылка вида:
```
https://ВАШ_ДОМЕН/ВАШ_SLUG
```

## Примечания по безопасности
- Все данные хранятся в `data/users.json`.
- Доступ к админке осуществляется по «секретной» ссылке `/a/SLUG`.
- Для продакшена рекомендуется добавить полноценную авторизацию.
