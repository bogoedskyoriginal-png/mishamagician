# Магическое предсказание — инструкция

## Установка Node.js
1. Скачайте и установите Node.js с официального сайта: https://nodejs.org/

## Запуск проекта локально
1. Распакуйте папку проекта.
2. Откройте терминал в папке проекта.
3. Установите зависимость:
   ```bash
   npm install express
   ```
4. Запустите сервер:
   ```bash
   node server.js
   ```

Сервер по умолчанию слушает порт `8080`. Можно поменять, задав переменную окружения `PORT`.

## Как узнать IP-адрес компьютера
1. Откройте PowerShell.
2. Выполните команду:
   ```bash
   ipconfig
   ```
3. Найдите строку `IPv4-адрес` у вашего активного адаптера (Wi‑Fi).

## Как открыть страницу зрителя (локально)
1. На телефоне зрителя откройте в браузере:
   ```
   http://IP_КОМПЬЮТЕРА:8080
   ```
2. Добавьте страницу на домашний экран (PWA).

## Как открыть админ-панель (локально)
1. На своём iPhone откройте:
   ```
   http://IP_КОМПЬЮТЕРА:8080/admin.html
   ```

## Как пользоваться (админка)
1. Введите 5 предметов — именно эти названия будут использоваться в поиске Google Картинок.
2. Зажмите область ввода на 3 секунды.
3. Тапните 1–5 раз — число тапов выбирает предмет.
4. Команда сразу уходит на сервер.

## Публичный доступ (бесплатный тест)
Нужен хостинг, где сервер будет доступен из любых сетей.

### Вариант A: Render (самый простой)
Ограничения бесплатного плана:
- Сервис уходит в сон после 15 минут без запросов.
- На пробуждение может уходить до 1 минуты.
- Лимит 750 часов работы в месяц на все бесплатные сервисы.

Шаги:
1. Создайте репозиторий на GitHub и загрузите в него этот проект.
2. Зайдите на Render и создайте **Web Service** из репозитория.
3. В настройках укажите команду запуска: `node server.js`.
4. Render сам задаст переменную `PORT`.
5. Получите ссылку вида: `https://ВАШЕ_ИМЯ.onrender.com`

### Вариант B: Railway (тоже можно)
- Бесплатный план даёт $1 кредит в месяц.
- Новым аккаунтам дают $5 на 30 дней, потом остаётся $1/мес.

## Как дать ссылку зрителю
Пример:
- Зритель: `https://ВАШ_ДОМЕН_ИЛИ_ХОСТИНГ`
- Админка: `https://ВАШ_ДОМЕН_ИЛИ_ХОСТИНГ/admin.html`

Можно дополнительно сократить ссылку через любой сервис коротких ссылок.
