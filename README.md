# Магическое предсказание — инструкция

## Установка Node.js
1. Скачайте и установите Node.js с официального сайта: https://nodejs.org/

## Запуск проекта локально
1. Распакуйте папку проекта.
2. Откройте терминал в папке проекта.
3. Установите зависимость:
   ```bash
   npm install express
   ```
4. Запустите сервер:
   ```bash
   node server.js
   ```

Сервер по умолчанию слушает порт `8080`. Можно поменять, задав переменную окружения `PORT`.

## Публичный доступ (Render)
После деплоя ваш домен будет вида:
```
https://ВАШЕ_ИМЯ.onrender.com
```

## Главная админка (создание пользователей)
Главная админка доступна по адресу:
```
https://ВАШ_ДОМЕН/master
```

Логин/пароль по умолчанию:
- Логин: `master`
- Пароль: `master123`

В главной админке можно создавать пользователей с собственными ссылками:
- Зритель: `https://ВАШ_ДОМЕН/u/ВАШ_SLUG`
- Админка: `https://ВАШ_ДОМЕН/a/ВАШ_SLUG`

## Дефолтный пользователь
Создаётся автоматически:
- Зритель: `https://ВАШ_ДОМЕН/u/default`
- Админка: `https://ВАШ_ДОМЕН/a/default`

Логин/пароль админки по умолчанию:
- Логин: `admin`
- Пароль: `admin123`

## Как пользоваться (админка)
1. Войдите в админку своего пользователя.
2. Нажмите ⚙ и задайте 5 предметов.
3. Зажмите экран на 3 секунды.
4. Тапните 1–5 раз — отправится выбор.
5. Для сброса нажмите кнопку «Сброс».

## Как открыть страницу зрителя
Зрителю отправляется ссылка вида:
```
https://ВАШ_ДОМЕН/u/ВАШ_SLUG
```

## Примечания по безопасности
- Логины и пароли хранятся в файле `data/users.json`.
- Для реального продакшена рекомендуется шифрование/хэширование паролей и отдельная база.
